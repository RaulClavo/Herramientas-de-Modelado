# Representador Diedrico
Con este programa (Representador Diedrico.py) podrás representar coordenadas diédricas tanto en 4 como en 2 cuadrantes.
## Instrucciones
-  Define tus "CUADROS" y tu "ANCHO" para establecer la escala y resolución de las gráficas.
-  Define tus coordenadas a representar dentro de la lista "coords" con forma (Nombre, Desplazamiento, Alejamiento, Cota).
-  Ejecuta el programa y nota que tus coordenadas han sido representadas en 2 y 4 cuadrantes diétricos.
