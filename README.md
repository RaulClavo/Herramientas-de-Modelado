# Herramientas de Modelado

Este repositorio contiene dos herramientas gráficas escritas en Python para la asignatura de "Simulación y Modelado"

---

## Pintor Isométrico

Con este programa podrás crear, colorear y dividir celdas dentro de una cuadrícula isométrica, haciendo fácil representar estructuras tridimensionales en proyección isométrica.

### Instrucciones

- En la parte inicial del código puedes definir tus celdas y el tamaño de las mismas.
- Utiliza el click izquierdo para pintar el triángulo seleccionado con el color activo.
- Utiliza el click derecho para dividir el triángulo por el eje seleccionado.

---

## Representador Diédrico

Con este programa podrás representar de forma sencilla coordenadas diédricas tanto en 4 como en 2 cuadrantes.

### Instrucciones

- En la parte inicial del código puedes definir la escala y resolución de las gráficas.
- Define las coordenadas a representar dentro de la lista `coords` con forma `(Nombre, Desplazamiento, Alejamiento, Cota)`.
- Ejecuta el programa y verás tus coordenadas representadas en 2 y 4 cuadrantes diédricos.
